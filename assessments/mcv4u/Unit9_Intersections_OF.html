📕 Answer Key

1. \(1+2t=5-s\) and \(2+3t=1+2s\). From first: \(s=4-2t\); sub into second: \(2+3t=1+2(4-2t)=9-4t\) → \(7t=7\) → \(t=1\), \(s=2\). Point: \((3,5)\).

2. Same direction \((2,1,-1)\). Test if \((3,1,0)\) on \(L_1\): \(3=1+2t\) → \(t=1\); then \(y=0+1=1\) ✓; \(z=1-1=0\) ✓. → coincident.

3. \((2+t)+(-1+2t)+2(3-t)=8\) → \(2+t-1+2t+6-2t=8\) → \(7+t=8\) → \(t=1\). Point: \((3,1,2)\).

4. \(\vec d=\vec n_1\times\vec n_2=(1,2,-1)\times(2,-1,1)=(2\cdot 1-(-1)(-1),\,(-1)(2)-(1)(1),\,(1)(-1)-(2)(2))=(1,-3,-5)\). Set \(z=0\): \(x+2y=4,\ 2x-y=3\) → \(y=(2x-3)\), so \(x+2(2x-3)=4\) → \(5x=10\) → \(x=2,y=1\). Line: \((2,1,0)+t(1,-3,-5)\).

5. Unique point ↔ rank 3 → (b).

6. Add (1)+(3): \(2x+5y=4\). (1)·2 - (2): \(0+3y-z=...\) easier: from (1) \(z=6-x-y\); sub into (2): \(2x-y+3(6-x-y)=14\) → \(-x-4y=-4\) → \(x+4y=4\). System: \(2x+5y=4,\ x+4y=4\). \(2(x+4y)-( 2x+5y)=8-4\) → \(3y=4\) → \(y=4/3\), \(x=4-16/3=-4/3\), \(z=6-(-4/3)-4/3=6=18/3\). Solution \((-4/3,4/3,6)\). Verify (3): \(-4/3+16/3-6=12/3-6=4-6=-2\) ✓.

7. \(\vec d_1\times\vec d_2=(1,1,1)\times(2,3,1)=(1\cdot1-1\cdot3,\,1\cdot2-1\cdot1,\,1\cdot3-1\cdot2)=(-2,1,1)\). \(\vec{P_1P_2}=(1,-1,1)\). Dot: \(-2-1+1=-2\). \(|\vec d_1\times\vec d_2|=\sqrt{6}\). Distance \(=2/\sqrt 6\approx 0.816\). Skew because \((-2)\ne 0\).

8. \(\pi_2=2\pi_1\) (same plane). \(\pi_3\) intersects \(\pi_1\) in a line (different normal). System reduces to 2 effective equations in 3 unknowns → infinitely many solutions along that line.

9. (a) Set up 2 eqs in 2 params \(s,t\), solve. (b) Same, but check consistency in 3rd component (otherwise skew or parallel). (c) Substitute parametric line into plane scalar equation, solve for \(t\). (d) Solve 2-eq 3-unknown system: assign one variable as parameter.

10. Cases: (i) Single point — three planes mutually intersecting. (ii) Common line — two planes coincident plus one intersecting; or all three meeting along the line. (iii) Coincident — all three same. (iv) No common point — parallel planes (one or more), or "triangular prism" (each pair meets in a line, no common line).

11. Two non-parallel normals span the orthogonal complement direction. The intersection set satisfies both linear equations → a 1-parameter family (line). Direction perpendicular to both normals = \(\vec n_1\times\vec n_2\).

12. \(\vec d_1\times\vec d_2\) is perpendicular to both lines' directions. Project the displacement \(\vec{P_1P_2}\) onto this common-perpendicular direction → shortest distance.

13. Set positions equal: \(2t=5\), \(3t=3s\), \(1+0.1t=1\). From 1st: \(t=2.5\); from 3rd: \(t=0\). Contradiction → paths don't even cross at same place → no intersection of paths in 3D (skew). No collision.

14. \((1+2t)+2(2+3t)-(5-4t)=10\) → \(1+2t+4+6t-5+4t=10\) → \(12t=10\) → \(t=5/6\). Point: \((1+5/3,2+5/2,5-10/3)=(8/3,9/2,5/3)\).

15. Try eliminating: (1)+(2): \(3x+0+0=6\) → \(x=2\). Sub into (1): \(4+y-z=4\) → \(y=z\). Sub into (3): \(6+2y-2y=6\) ✓ identically. So infinitely many solutions: \((2,t,t)\). Three planes meet in a line — common safe corridor exists.
